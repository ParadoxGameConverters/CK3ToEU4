##### INSTRUCTIONS
# CK3 Ethos and Traditions map to EU4 National Traditions and Ideas/Ambitions respectively.
# There are three fields to a link, a ck3 = {} part, an eu4 = {} part and an optional rule = {} part.
# The ck3 field determines whether an ethos or tradition is being mapped. ck3 = { ethos = ethos_name } or ck3 = {tradition = tradition_name }.
# The eu4 field determines which EU4 modifer effects to use in the generated National Traditions/Ideas/Ambitions.
# The rule field determines when a culture should have different EU4 ideas from the same base tradition, this is only for special cases.

### Base Mappings
# These are links without a rule = {} field. There should only be ONE per ethos/tradition.
# Mods that add ethoses or traditions will need new base mappings defined in order to be handled properly.

### Rule Mappings
# These are links with the rule = {} field. There may be many per ethos/tradition.
# If the rule given is true for the culture generating ideas, the base mapping will be overridden with the effects associated with that link.

# Ex: 
# link = { ck3 = { tradition = tradition_seafaring } eu4 = { naval_forcelimit_modifier = 0.1 } }
# link = { ck3 = { tradition = tradition_seafaring } rule = { heritage = heritage_north_germanic } eu4 = { may_explore = yes } }

# In this case all cultures with the CK3 seafaring tradition will get an EU4 idea that boosts naval forcelimit. Unless that CK3 culture had heritage_north_germanic,
# then the generated EU4 idea would grant Quest for the New World.

# Other things to note about rules:
# The order of rule mappings in a file is irrelavent to the order in which they are applied.
# Multiple rules of different types can be combined to make more specific rules.
# Ex: rule = { heritage = heritage_north_germanic ethos_trigger = ethos_bellicose }.
# Rules have a precedence order shown at the bottom of instructions. This means if both rules are true, only the higher precedence rule applies.
# More specific rules have higher precedence than less specific rules, when they start with the same precedence.
# ALL links will use the CK3 localization to generate the EU4 Idea name and desctription localization, unless you specify an idea name.
# Ex: link = { ck3 = { tradition = tradition_one } rule = { heritage = heritage_red idea_name = my_idea_name } eu4 = { puppies = yes } }
# If idea_name is specified, the converter expects manual localizations for my_idea_name and my_idea_name_desc in the localisation/replace/tradition_ideas_l_language.yml file of blankMod.

### Currently Supported Rules - (In order of precedence)
# 0: heritage = heritage_north_germanic (Rule applies if the culture has the given heritage)
# 1: ethos_trigger = ethos_bellicose (Rule applies if the culture has the given ethos)
