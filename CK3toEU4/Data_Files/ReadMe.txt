This program will convert a Crusader Kings III save game to a Europa Universalis IV save game.

----
IMPORTANT LINKS
----

Forum Release Thread: https://forum.paradoxplaza.com/forum/threads/ck3-to-eu4-converter-release-thread.1415137/
Forum Development Thread: https://forum.paradoxplaza.com/forum/threads/ck3-to-eu4-converter-development-thread.1415140/
Github project: https://github.com/ParadoxGameConverters/CK3toEU4
Windows development build: https://github.com/ParadoxGameConverters/CK3toEU4/releases/tag/windows_development_build

----
REQUIREMENTS
----
CK3 - 1.0
EU4 - 1.30

-----
INSTRUCTIONS
-----
1. Run ConverterFrontend.exe.
2. Fill out all fields, go through the option tabs, and finally click Convert.

----
MANUAL METHOD INSTRUCTIONS
----

1. Download the converter and unzip it wherever.
2. Copy your CK3 save into the converter directory.
3. Edit configuration-example.txt to point at your game installs, your savegame location, and to set any options you care about, and save it as configuration.txt.
4. Run CK3ToEU4Converter.exe
5. Go into the output folder, then the subfolder named after your save.
6. Copy the folder named after your save and the corresponding mod file into your EU4 mod folder.
7. Start EU4, and select the mod.
